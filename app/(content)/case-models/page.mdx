## zudo-block-60-open

<CatchText>エントリーレベルのフレームキット。</CatchText>

当ショップでご好評をいただいているエントリークラスのフレームキットです。必要最小限のパーツだけでモジュールを固定できるため、初めての方やDIY派の方でも気軽にモジュラーシンセサイザーを始められます。価格もリーズナブルで、入門機として最適です。

### 主な特徴

- **ミニマル構成**: モジュール取り付けに必要な部品のみを含む
- **オープンデザイン**: ケースなし、フレームのみの構造
- **パネルオプション**: Type A（ストレート）とType B（角度付き）の2種類
- **簡単な組み立て**: サイドパネルとアルミフレームをネジで接続

### 対象ユーザー

モジュラーシンセサイザー初心者や、高額な商用ケースを購入する前に手頃な価格で試したいDIY愛好家に最適です。

### カラーカスタマイズ対応

従来は3Dプリント製のみでしたが、新たにカラーを選べるアクリル版を追加しました。さらに、後からフルケースのzudo-block-60へ発展させられるアップグレードキットも別売しています。

**参考記事**: [zudo-block-60-open紹介記事](https://takazudomodular.com/notes/2025-04-17-zb60-open-intro/)

### 利用可能なバリエーション

- zudo-block-60-open-ACR-A（アクリル Type A）
- zudo-block-60-open-ACR-B（アクリル Type B）
- zudo-block-60-open-ACR-upgrade（アップグレードキット - アクリル）
- zudo-block-60-open-3DP-A（3Dプリント Type A）
- zudo-block-60-open-3DP-B（3Dプリント Type B）
- zudo-block-60-open-3DP-upgrade（アップグレードキット - 3Dプリント）

## zudo-block-60

**スタンダードサイズの透明アクリルケース**

### 概要

60HP幅のクリアアクリルケースDIYキットです。最初に40HPモデルを開発しましたが、「もう少し余裕がほしい」というユーザーの声に応え、より汎用性の高い60HP版をリリースしました。現在はこちらが当ブランドのスタンダードサイズとなっています。

透明なアクリル素材により、モジュールのLED照明やバスボードの光が美しく拡散し、サイバーパンク的な美しさを演出します。フロントパネルはお好みのカラーにカスタマイズ可能です。自分だけのケースを組み立てながら、将来のシステム拡張にも柔軟に対応できます。

### 主な特徴

- **ビジュアル性**: 透明アクリルデザインがモジュール内部やLED照明を美しく見せる
- **角度調整機能**: 内部フレームを複数の角度に固定でき、演奏時の視認性と操作性を向上
- **レールオプション**: 予算に応じて3種類から選択可能
  - **Lite**: ASAプラスチックレール（最も手頃で軽量）
  - **Dual**: ASAレールとアルミナット（バランス型）
  - **Metal**: フルアルミ構造（プレミアム）
- **カスタマイズ**: パネルカラーとサイドパネルスタイル（Type AまたはType B）を選択可能

### 仕様

- **サイズ**: 60HPフォーマット
- **奥行き**: 約8cm
- **構造**: レーザーカットアクリルパネルと金属ハードウェア
- **価格帯**: ¥8,980〜¥16,980（レール選択により変動）

### 拡張性

zudo-railシステムを使用して複数のzudo-blockユニットを接続でき、セットアップを拡張しながら統一感のある外観を維持できます。

**参考記事**: [zudo-block-60紹介記事](https://takazudomodular.com/notes/2025-01-23-zb60-intro/)

### 利用可能なバリエーション

- zudo-block-60-ACR-A（アクリル Type A）
- zudo-block-60-ACR-B（アクリル Type B）
- zudo-block-60-3DP-A（3Dプリント Type A）
- zudo-block-60-3DP-B（3Dプリント Type B）

## zudo-block-40

**コンパクトな40HPアクリルケース**

### 概要

40HP幅のアクリルケースで、zudo-blockシリーズ最初のモデルです。手軽に持ち運べるコンパクトさながら、モジュール運用に必要な機能をしっかり備えています。初めての一台としても安心してお選びいただけます。

### 主な特徴

- **ビジュアル性**: 透明アクリルデザインがシンセサイザーの内部コンポーネントとLED照明を美しく見せる
- **角度調整機能**: サイドパネルの複数の穴を使用して内部フレームをさまざまな角度に固定可能
- **3つのレールオプション**:
  - **Lite**（¥7,980）: ASAプラスチックレール - 予算重視だが慎重な扱いが必要
  - **Dual**（¥11,980）: プラスチックレールとメタルナットのハイブリッド設計
  - **Metal**（¥15,980）: 耐久性に優れたプレミアムアルミ構造
- **カスタマイズオプション**: Red、Orange、Greenの3色とType A（ストレート）、Type B（ステップ）の2種類のサイドパネルから選択可能

### 仕様

標準的なEurorackモジュールに対応し、奥行きは約8cmです。フラットパック形式で出荷され、基本工具（六角レンチ、ドライバー、スパナ）で組み立てできます。

### 拡張性

複数のユニットを組み合わせてセットアップを拡張できるモジュラー設計です。

**参考記事**: [zudo-block-40紹介記事](https://takazudomodular.com/notes/2024-11-13-zb40-intro/)

### 利用可能なバリエーション

- zudo-block-40-ACR-A（アクリル Type A）
- zudo-block-40-ACR-B（アクリル Type B）
- zudo-block-40-3DP-A（3Dプリント Type A）
- zudo-block-40-3DP-B（3Dプリント Type B）

## 10BOX

**蓋付きケースモデル**

### 概要

フタ付きで持ち運びに便利な最新モデルです。移動時の衝撃やホコリからモジュールを守りつつ、すぐに演奏を始められる手軽さが魅力です。104HPの大容量で、多くのモジュールを収納できます。

### 主な特徴

- **蓋付きデザイン**: 持ち運びと保管時にモジュールを保護
- **大容量**: 104HPの十分なスペース
- **2つの深さオプション**: Shallow（浅型）とDeep（深型）から選択可能
- **3Dプリントバージョン**: 現在は3Dプリントバージョンのみ提供

### 仕様

- **サイズ**: 104HPフォーマット
- **バージョン**: Shallow（浅型）とDeep（深型）
- **材質**: 3Dプリントのみ

### 利用可能なバリエーション

- 10BOX-Shallow-3DP（浅型 - 3Dプリント）
- 10BOX-Deep-3DP（深型 - 3Dプリント）

すべてのケースモデルは[設定ページ](/m)でカスタマイズできます。パネルカラーの選択、レールタイプの選択、リアルタイムの価格見積もりが可能です。
